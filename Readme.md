
# Auth Manager 

This project aim to create service to handle Auth related work. 

Supports following routes 

- Un protected routes 
    - [ ] login 
    - [ ] register
- Protectec routes 
    - [ ] manage users
    - [ ] manage roles
    - [ ] manage permissions
    - [ ] manage roles <> permissions
    - [ ] manage roles <> users
    - [ ] verify permissions
